DOCUMENTAZIONE POST-PROCESSING SCRIPTS

Questi script vengono eseguiti al termine della generazione dei chart (dopo che tutti i livelli Easy, Medium, Hard sono stati creati).
Servono a rifinire il file .sm, pulire errori comuni e organizzare i file finali.

ORDINE DI ESECUZIONE:
1. PP_mute.py
2. PP_IntroEnd.py
3. PP_azioniFinali.py (eseguito alla fine assoluta della pipeline)

--------------------------------------------------------------------------------
1. PP_mute.py (Ex chart_refiner_mute.py)
--------------------------------------------------------------------------------
Scopo:
Rimuove le note (tap, hold, roll, jump) nelle sezioni dove il volume audio è troppo basso (silenzio).

Funzionamento:
- Legge "analysis_data.json" per ottenere i livelli di volume (RMS) beat per beat.
- Calcola una soglia di silenzio (15% del volume medio globale della canzone).
- Scansiona ogni riga di ogni chart nel file .sm.
- Se il volume audio in quel punto è sotto la soglia, elimina le note (sostituisce la riga con "0000").
- Utile per evitare che vengano piazzate note su parti silenziose o rumore di fondo.

--------------------------------------------------------------------------------
2. PP_IntroEnd.py (Ex chart_refiner_IntroEnd.py)
--------------------------------------------------------------------------------
Scopo:
Gestisce l'intro e l'outro della canzone per garantire un inizio e una fine puliti.

Funzionamento:
- Legge "analysis_data.json".
- Determina:
  - First Active Beat: quando inizia davvero la musica (RMS > 0.02).
  - Last Active Beat: quando finisce la musica.
  - Fade Out Start: quando inizia la sfumatura finale (RMS < 0.15).
- Regole applicate:
  - Prima del First Active Beat: Rimuove TUTTE le note.
  - Dopo il Last Active Beat: Rimuove TUTTE le note.
  - Durante il Fade Out: Mantiene solo le note sui battiti principali (Red notes), rimuove tutto il resto (Blue, Yellow, ecc.) per accompagnare la sfumatura.

--------------------------------------------------------------------------------
3. PP_azioniFinali.py (Ex azioni_finali.py)
--------------------------------------------------------------------------------
Scopo:
Organizzazione finale dei file e pulizia della directory di lavoro.

Funzionamento:
- Legge il file .sm per trovare il tag #MUSIC: (nome del file audio).
- Crea una cartella con il nome della canzone (basato sul file audio).
- Sposta all'interno della cartella:
  - Il file .sm
  - Il file .ssc (se presente)
  - Il file .mp3
  - Le immagini di background (BG.png) e banner (BN.png) se presenti.
- Copia "analysis_data.json" nella cartella per riferimento futuro.
- Questo script lascia la root pulita e prepara la canzone per essere copiata nella cartella Songs di StepMania.
